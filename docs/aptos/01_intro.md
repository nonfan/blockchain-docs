# 公链概览

**Aptos 是一个高性能、可扩展的去中心化 Layer 1 区块链平台**，旨在解决现有区块链技术在吞吐量、延迟和可扩展性方面的限制，特别适用于大规模的去中心化应用（DApps）和企业级应用。它由曾在 Meta（前身为 Facebook） 的 Diem 项目团队开发，并且融合了新型的共识机制和编程语言设计理念。其目标是成为一个具有高吞吐量、低延迟、高安全性和强大可扩展性的区块链平台。

## 核心技术亮点
| 技术点           | 说明                                                                 |
|------------------|----------------------------------------------------------------------|
| 编程语言         | Move（资源导向，内置正式验证 Move Prover）                           |
| 共识机制         | AptosBFT（HotStuff 变种） + PoS，亚秒级最终性                        |
| 并行执行引擎     | Block-STM（乐观并行 + 自动重试），理论 TPS > 160,000                 |
| 交易费用         | 平均 ~0.0005 USD，极低                                              |
| 状态同步         | 节点恢复/新节点加入极快                                              |
| 合约升级         | 无缝热升级，用户无感知                                               |
| 稳定性           | 主网上线以来 0 宕机记录                                              |

- **解锁节奏**：每月线性解锁约 1100-1200 万枚（2025 年仍有一定抛压，但市场已逐步消化）

## 优势 vs 劣势
### 优势
- 性能与安全性双高，真正机构友好
- 交易费用极低、确认速度快，适合支付/游戏/高频 DeFi
- 2025 年 RWA 与稳定币落地速度最快公链之一
- 合作伙伴：BlackRock、Microsoft、Google Cloud、Aave、Tether 等

### 劣势
- 验证者数量相对较少，去中心化程度仍有提升空间
- Move 语言学习成本较高（但工具链已大幅完善）
- 代币仍存在线性解锁抛压

## 未来展望（2026+）
- 横向扩展计划 Shardines → 目标 50 万+ TPS
- 更多传统金融资产（国债、基金、股票等）上链
- Move 生态与 EVM 互操作性进一步增强
- 继续巩固“全球交易引擎 + 机构级 RWA 平台”定位
